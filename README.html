elixir-mongo

A MongoDB driver in Elixir.

Provides wrappers for CRUD and Aggregate operations:

  • find
  • insert
  • update
  • remove
  • count
  • distinct
  • mr (mapReduce)
  • group
  • aggregate

It is very light on admin commands

  • getlasterror
  • drop

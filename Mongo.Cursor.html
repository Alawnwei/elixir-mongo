stream(c)

Creates a stream of documents for the given the cursor

stream, exectutes the query if not done yet, then, streams documents one by one (retreived from DB by batch). When exhausted, it kills the cursor.

Source

toArray(find)

Creates a list of document retreive by a find query

toArray, exectutes the query then accumulate in a list all documents one by one. It calls Mongo.Cursor/1 then pipes the resulting stream to Enum.to_list/1
